# Sounds Directory

This directory contains audio files for the music player feature. Audio files should be in MP3 or OGG format for browser compatibility.

## Directory Structure

- `ambient/` - Ambient background sounds
- `nature/` - Nature sounds
- `focus/` - Focus and concentration sounds

## Required Audio Files

### Ambient Sounds
- `white-noise.mp3` - White noise for focus
- `brown-noise.mp3` - Brown noise (deeper, warmer)
- `pink-noise.mp3` - Pink noise (balanced frequency)
- `coffee-shop.mp3` - Coffee shop ambiance

### Nature Sounds
- `rain.mp3` - Rain sounds
- `forest.mp3` - Forest ambiance
- `ocean-waves.mp3` - Ocean waves
- `thunderstorm.mp3` - Thunderstorm sounds
- `birds.mp3` - Birds chirping

### Focus Sounds
- `binaural-beats.mp3` - Binaural beats for focus
- `deep-focus.mp3` - Deep focus music
- `concentration.mp3` - Concentration music

## Adding Audio Files

1. Place audio files in the appropriate category directory
2. Use lowercase filenames with hyphens (e.g., `white-noise.mp3`)
3. Ensure files are in MP3 or OGG format
4. Keep file sizes reasonable (under 5MB per file recommended)
